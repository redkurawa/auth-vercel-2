Auth Vercel is a Google OAuth 2.0 authentication application that can run on localhost or be deployed to Vercel. This application features a CSS toggle to switch between basic and modern display modes. Users can log in with their Google account and view their profile after successful authentication.

Auth Vercel adalah aplikasi autentikasi Google OAuth 2.0 yang dapat dijalankan di localhost maupun di-deploy ke Vercel. Aplikasi ini memiliki fitur toggle CSS untuk beralih antara tampilan basic dan modern. Pengguna dapat login dengan akun Google mereka dan melihat profil mereka setelah berhasil login.
