What is DOM?

Apa itu DOM?

The Document Object Model (DOM) is a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content. DOM treats an XML or HTML document as a tree structure, where each node is an object representing a part of the document.

Document Object Model (DOM) adalah antarmuka pemrograman untuk dokumen web. Ini merepresentasikan halaman agar program dapat mengubah struktur, gaya, dan konten dokumen. DOM memperlakukan dokumen XML atau HTML sebagai struktur pohon, di mana setiap node adalah objek yang mewakili bagian dari dokumen.

Why DOM with Google Auth?

Mengapa DOM dengan Google Auth?

  • Dynamic UI Updates — Update the interface instantly when a user logs in or out without reloading the page.
  • Perbarui UI Secara Dinamis — Perbarui antarmuka secara instan saat pengguna login atau logout tanpa memuat ulang halaman.
  • User Experience — Create smooth, responsive interactions that feel modern and professional.
  • Pengalaman Pengguna — Buat interaksi yang halus dan responsif yang terasa modern dan profesional.
  • State Management — Manipulate DOM elements based on authentication state (logged in vs logged out).
  • Manajemen State — Manipulasi elemen DOM berdasarkan status autentikasi (sudah login vs belum login).

Code Example

Contoh Kode

Here's a simple example of manipulating DOM based on Google Auth state:

Berikut adalah contoh sederhana manipulasi DOM berdasarkan status Google Auth: 

// Check if user is authenticated
const isAuthenticated = typeof user !== 'undefined';

// DOM elements to manipulate
const loginBtn = document.getElementById('login-btn');
const profileSection = document.getElementById('profile');
const userName = document.getElementById('user-name');

// Update UI based on auth state
if (isAuthenticated) {
  loginBtn.style.display = 'none';
  profileSection.style.display = 'block';
  userName.textContent = user.displayName;
} else {
  loginBtn.style.display = 'block';
  profileSection.style.display = 'none';
}
